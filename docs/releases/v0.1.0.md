# Release notes — v0.1.0

**Tag:** `v0.1.0`  
**Package version:** `0.1.0` (`pyproject.toml`, `agent_llm_wiki_matrix.__version__`, `alwm version`)

This is the first **semver-tagged** release. The focus is a **reproducible, offline-first** toolkit: typed JSON artifacts, deterministic pipelines, a benchmark harness with registry-backed prompts, and Docker/Compose/Buildx workflows. Live LLM services and browser automation are **optional** and validated outside default CI.

## Highlights

- **CLI (`alwm`)** for validation, wiki-style pipelines, benchmarks, and prompt registry inspection.
- **No network required** for `just ci` or the primary example flows (fixture/mock modes).
- **Multi-arch container builds** via Buildx Bake (`linux/amd64`, `linux/arm64`) for the main `runtime` image.

## Install

- **Local:** Python **3.11+**, **[uv](https://docs.astral.sh/uv/)** — `uv venv`, `uv pip install -e ".[dev]"`, run commands with `uv run …` (see [README.md](../../README.md) quickstart and **`AGENTS.md`**).
- **Container:** build the `runtime` target from the `Dockerfile` or use `docker buildx bake` (see [docs/release-readiness.md](../release-readiness.md#minimum-supported-environments)).
